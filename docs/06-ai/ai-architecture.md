# Architecture IA

## Principe
Le LLM n’est pas le cerveau. Il est une couche de langage.

## Composants
Financial Memory, Decision Engine, Recommendation Engine, Simulation Engine, Forecast Engine, Risk Engine, Opportunity Engine, Explainability Engine, Confidence Engine, LLM Gateway.

## Règles
Le LLM ne voit jamais toute la base. Les calculs critiques sont déterministes. Chaque recommandation est traçable et possède un score de confiance.
