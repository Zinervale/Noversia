# Manifeste produit

1. Les données appartiennent toujours à l'utilisateur.
2. L'IA explique toujours ses recommandations.
3. L'utilisateur décide toujours.
4. L'IA n'invente jamais.
5. Le produit doit créer une valeur financière mesurable.
6. Chaque écran doit aider à prendre une décision.
7. La mémoire financière est un actif utilisateur.
8. Aucun partenariat ne doit biaiser les recommandations.
