# US-001 — Importer un fichier CSV bancaire

## En tant que
Utilisateur

## Je veux
Importer un fichier CSV de transactions

## Afin de
Construire mon historique financier sans connecter ma banque.

## Critères d’acceptation
- Le système accepte un CSV.
- Le système détecte les colonnes principales.
- Le système affiche un aperçu.
- Le système signale les lignes invalides.
- Le système importe les lignes valides.
