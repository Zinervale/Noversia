# US-002 — Créer un compte manuel

## En tant que

Utilisateur

## Je veux

Créer un compte financier manuel

## Afin de

Suivre un compte non synchronisé.

## Critères d'acceptation

- Le compte possède un nom.
- Le compte possède un type.
- La devise par défaut est EUR.
- Le solde initial peut être saisi.
- Le compte apparaît dans la liste des comptes.
