# US-003 — Prévisualiser un import CSV

## En tant que

Utilisateur

## Je veux

Voir un aperçu des transactions avant import définitif

## Afin de

Vérifier que le fichier est correctement interprété.

## Critères d'acceptation

- Les 10 premières lignes sont affichées.
- Les colonnes détectées sont affichées.
- L'utilisateur peut confirmer ou annuler.
- Les erreurs de format sont visibles.
