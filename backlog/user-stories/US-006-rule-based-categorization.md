# US-006 — Catégoriser automatiquement par règles

## Statut

Réalisé en v6.

## Critères d'acceptation

- Le système applique les règles actives.
- Les règles sont priorisées.
- Une transaction importée peut recevoir une catégorie.
- Le score de confiance est conservé.
- Les règles sont consultables via API.
