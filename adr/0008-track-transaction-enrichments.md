# ADR 0008 — Historiser les enrichissements de transactions

## Statut

Accepté

## Contexte

Les corrections utilisateur sont une source d'apprentissage importante.

## Décision

Chaque correction ou enrichissement est stocké dans `transaction_enrichments`.

## Conséquences

- Traçabilité.
- Possibilité d'audit.
- Base pour l'apprentissage futur.
